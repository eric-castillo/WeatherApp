## Weather App

[![N|Solid](https://cldup.com/dTxpPi9lDf.thumb.png)](https://nodesource.com/products/nsolid)

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

Autor: Castillo Ruiz Eric Fabian

## Funcionamiento

- App de Weather que permite ver el Clima de ciertas areas del planeta.
- Se usa useEffect y una API de Clima para poder tener todos los datos de busqueda

## Instalacion

Necesita [Node.js](https://nodejs.org/).

Instala las Dependencias e Inicia el Servidor (Dentro de la terminal del Programa)

```sh
npm i
npm run dev
```
